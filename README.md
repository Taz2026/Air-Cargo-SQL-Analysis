# Air-Cargo-SQL-Analysis
SQL-based analysis of air cargo operations to evaluate delays, capacity utilization, route revenue, and service performance. Demonstrates data modeling, analytical querying, and supply chain performance insights using structured airline cargo datasets.

🛠 Tools Used
* SQL (PostgreSQL)
* GitHub
* Data modeling concepts

🔎 Key Insights
* Identified average delay trends by airline
* Measured capacity utilization rates
* Evaluated route-level revenue performance
* Calculated on-time delivery performance

📊 Business Impact
* Supports KPI-driven supply chain decisions
* Highlights delay patterns impacting service levels
* Identifies high-revenue trade lanes
* Enables operational efficiency improvements
